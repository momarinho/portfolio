# Retro RPG Portfolio - Mateus Marinho

A personal portfolio website with a 16-bit RPG theme, featuring specific quests (projects), skill trees, and sound effects. Built with Next.js 15, TypeScript, and Tailwind CSS v4.

## 🎮 Features

- **Retro Aesthetic**: Pixel art fonts (Press Start 2P, VT323), glowing neons, and dark high-contrast theme.
- **Interactive Audio**: Custom hook for retro sound effects (beep, confirm, hover). Audio must be enabled by the user via the "Start Screen".
- **Quest Log system**: Project cards styled as RPG quests.
- **Responsive Design**: Mobile-friendly HUD and Navigation.

## 🛠️ How to Run

1. **Install Dependencies**:
   ```bash
   npm install
   ```
   *Note: Use `pnpm` or `yarn` if preferred.*

2. **Run Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📝 Customization

### Changing Content
All content is hardcoded in the component files for simplicity. You can edit them directly:

- **About Me / Stats**: Edit `src/components/CharacterSheet.tsx`.
- **Skills**: Edit `src/components/SkillTree.tsx`.
- **Projects**: Edit the `projects` array in `src/components/Projects.tsx`. Update the `link` fields to point to your real GitHub repositories.
- **Services**: Edit `src/components/Services.tsx`.

### Adding/Removing Projects
Open `src/components/Projects.tsx`:
```typescript
const projects = [
  {
    id: "new-project",
    title: "New Amazing Project",
    // ...
    link: "https://github.com/..."
  },
  // ...
]
```

### Audio
Sound logic is in `src/hooks/useRetroSound.ts`. You can tweak frequencies in `playOscillator` to change the "pitch" of the beeps.

## 🔮 Assumptions & Next Steps

- **Audio Policy**: Browsers block auto-playing audio. The "Start Screen" overlay serves as the user interaction required to unlock the `AudioContext`.
- **Assets**: No external image assets are used for icons to keep the repo lightweight. `lucide-react` icons are used. You can replace them with pixel-art SVGs in `src/assets` if desired.
- **Form**: The contact form in `GuildHall.tsx` is visual only. You should integrate it with a service like Formspree or EmailJS.
